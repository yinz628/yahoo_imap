# 产品概述

Yahoo Mail Extractor 是一个批量邮件数据提取工具，通过 IMAP 协议连接 Yahoo Mail 和 Gmail 账户，使用正则表达式提取邮件内容中的数据。

## 核心功能

- 通过 IMAP 协议连接邮箱账户（Yahoo、Gmail），使用应用专用密码
- 按文件夹、日期范围、发件人、主题筛选邮件
- 使用可配置的正则表达式从邮件内容中提取数据
- 导出提取结果到 CSV、Excel 或 SQLite 格式
- Web 界面用于交互式提取操作
- CLI 命令行用于自动化/脚本化提取

## 主要特性

- 多邮箱服务商支持（Yahoo、Gmail），自动检测
- 用户认证与会话管理
- 保存邮箱凭据（加密存储）和提取规则（按用户隔离）
- 连接健康监控与自动重连
- 批量处理，支持进度流式返回
- 规则历史记录，支持标签分类

## 目标用户

需要从大量邮件中程序化提取结构化数据（优惠码、订单号等）的用户。
